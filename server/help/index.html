Présentation

Contact est un logiciel de carnet d'adresse ecrit en PHP et MySQL, sa particularité est qu'il fonctionne sur un serveur web du type Apache et peut donc etre accessible en intranet ou via internet.

Installation

Configuration Necessaire:
  • Apache ou tout autre serveur supprotant le PHP et MySQL
  • PHP
  • MySQL
  • Konqueror ou tout autre Navigateur supportant le JavaScript et les evenements "OnClick" sur les images
Ma Configuration:
  • Apache v1.3.20
  • PHP v4.0.6
  • MySQL v11.15 distrib 3.23.41
  • Konqueror v2.2.1
Installation de Contact
1)Tout d'abord il vous faudra vous connecter en root
$ su
2) Lancer la configuration
# ./configure
3) Installer Contact:
# ./install-it

Se connecter à Contact




La premiere chose a faire est de se signer, il suffit d'entrer son code utilisateur et son mot de passe qui est stocké dans la table users.
L'administrateur permet d'ajouter ou supprimer des utilisateur. Pour se connecter en tant qu'administrateur il faut entrer le code utilisateur root ainsi que son mot de passe lui aussi stocké dans la table users.

La Liste des contacts




Une fois connecté, on se débranche automatiquement sur la liste des contacts.

On peu effectuer les actions suivantes:
  • Bouton Detail: Visualisation du détail d'un contact
  • Bouton Nouveau: Creation d'un nouveau contact
  • Bouton Chercher: Rechercher des contacts
  • Bouton Password: Modifier son mot de passe
  • Bouton Aide: Obtenir de l'aide sur la page
  • Bouton A Propos: Visualiser la boite A Propos du logiciel
  • Bouton Quitter: Se deconnecter de Contact

Créer un contact




Lorsque vous désirez créer un nouveau contact il est nécessaire de saisir au minimum deux champs à savoir le Nom et la Ville du contact.

Brève description des champs:
  • Nom: Il s'agit du nom du contact, ce champ est obligatoire
  • Prenom: Il s'agit du prénom du contact
  • Date de naissance: Date de naissance du contact
  • Pseudonyme: Il s'agit du pseudonyme du contact
  • Nature: Type de personne (Homme, Femme, Société)
  • Nature du contact: Famille, Amis, Connaissance, Professionnel
  • Adresse: Il s'agit de l'adresse du contact positionnée sur deux lignes
  • Code Postal: Il s'agit du code postal correspondant a la ville du contact
  • Ville: La ville ou reside le contact, ce champ est obligatoire
  • Tel Domicile: Le telephone personnel du contact
  • E-mail Personnel: L'adresse electronique personnelle du contact
  • Tel Bureaue: Le telephone professionnel du contact
  • E-mail Professionnel: L'adresse electronique professionnelle du contact
  • Tel Portable: Si le contact possède un portable indiquez son numéro ici
  • Fax: Si le contact possède un fax indiquez son numéro ici
On peu effectuer les actions suivantes:
  • Bouton Creation: Creation du contact
  • Bouton Aide: Obtenir de l'aide sur la page
  • Bouton Retour: Retour à la liste des contacts

Detail d'un contact




Cette page permet d'obtenir l'ensemble des informations a propos d'un contact en vue de les modifier, de supprimer le contact, ou tout simplement de les visualiser

  • Bouton Aide: Obtenir de l'aide sur la page
  • Bouton Retour: Retour à la liste des contacts

Chercher un contact




Cette page permet de selectionner certains criteres afin d'afficher une liste plus restreinte de contacts

Critères de recherche:
  • Nom: On peut saisir un nom complet ou partiel pour cela terminer la chaine par le caractère '%' ("B%" permet de chercher tous les contacts dont le nom commence par 'B')
  • Prenom: On peut saisir le prenom complet ou partiel pour cela terminer la chaine par le caractère '%'
  • Pseudonyme: On peut saisir le pseudonyme complet ou partiel pour cela terminer la chaine par le caractère '%'
  • Ville: On peut saisir la ville complete ou partielle pour cela terminer la chaine par le caractère '%'
  • Nature: On peut choisir entre Homme, Femme, Societe
  • Nature Contact: On peut choisir entre Famille, Amis, Connaissance, Profesionnel
  • Remarque: Si aucun critere n'est modifié la recherche renvera la liste complète des contacts
On peu effectuer les actions suivantes:
  • Bouton Chercher: Lancer la recherche avec les critères saisis
  • Bouton Aide: Obtenir de l'aide sur la page
  • Bouton Retour: Retour à la liste des contacts

Brève description des champs:
