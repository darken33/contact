Présentation

Contact est un logiciel de carnet d'adresse ecrit en PHP et MySQL, sa particularité est qu'il fonctionne sur un serveur web du type Apache et peut donc être accessible en intranet ou via internet.

Installation

Configuration Necessaire:
  • Apache ou tout autre serveur supportant le PHP et MySQL
  • PHP
  • MySQL
  • Konqueror ou tout autre Navigateur supportant le JavaScript et les évènements "OnClick" sur les images
Ma Configuration:
  • Apache v1.3.20
  • PHP v4.0.6
  • MySQL v11.15 distrib 3.23.41
  • Konqueror v2.2.1
Installation de Contact
1)Tout d'abord il vous faudra vous connecter en root
$ su
2) Lancer la configuration
# ./configure
3) Installer Contact:
# ./install-it
Mise a jour de Contact
1)Tout d'abord il vous faudra vous connecter en root
$ su
2) Lancer la configuration
# ./configure
3) Mettre à jour Contact:
# ./update-it

Se connecter à Contact




La première chose à faire est de se signer, il suffit d'entrer son code utilisateur et son mot de passe qui est stocké dans la table users.
L'administrateur permet d'ajouter ou supprimer des utilisateurs. Pour se connecter en tant qu'administrateur il faut entrer le code utilisateur root ainsi que son mot de passe lui aussi stocké dans la table users.

La Liste des contacts




Une fois connecté, on se débranche automatiquement sur la liste des contacts.

On peut effectuer les actions suivantes:
  • Bouton Détail: Visualisation du détail d'un contact
  • Bouton Nouveau: Création d'un nouveau contact
  • Bouton Chercher: Rechercher des contacts
  • Bouton Password: Modifier son mot de passe
  • Bouton Aide: Obtenir de l'aide sur la page
  • Bouton A Propos: Visualiser la boite A Propos du logiciel
  • Bouton Quitter: Se déconnecter de Contact

Créer un contact




Lorsque vous désirez créer un nouveau contact il est nécessaire de saisir au minimum deux champs à savoir le Nom et la Ville du contact.

Brève description des champs:
  • Nom: Il s'agit du nom du contact, ce champ est obligatoire
  • Prénom: Il s'agit du prénom du contact
  • Date de naissance: Date de naissance du contact
  • Pseudonyme: Il s'agit du pseudonyme du contact
  • Nature: Type de personne (Homme, Femme, Société)
  • Nature du contact: Famille, Amis, Connaissance, Professionnel
  • Adresse: Il s'agit de l'adresse du contact positionnée sur deux lignes
  • Code Postal: Il s'agit du code postal correspondant à la ville du contact
  • Ville: La ville où réside le contact, ce champ est obligatoire
  • Tel Domicile: Le téléphone personnel du contact
  • E-mail Personnel: L'adresse électronique personnelle du contact
  • Tel Bureaue: Le téléphone professionnel du contact
  • E-mail Professionnel: L'adresse électronique professionnelle du contact
  • Tel Portable: Si le contact possède un portable indiquez son numéro ici
  • Fax: Si le contact possède un fax indiquez son numéro ici
On peut effectuer les actions suivantes:
  • Bouton Création: Création du contact
  • Bouton Aide: Obtenir de l'aide sur la page
  • Bouton Retour: Retour à la liste des contacts

Détail d'un contact




Cette page permet d'obtenir l'ensemble des informations a propos d'un contact en vue de les modifier, de supprimer le contact, ou tout simplement de les visualiser

On peut effectuer les actions suivantes:

  • Bouton Supprimer: Supprimer le contact de la base de données
  • Bouton Aide: Obtenir de l'aide sur la page
  • Bouton Retour: Retour à la liste des contacts

Chercher un contact




Cette page permet de sélectionner certains critères afin d'afficher une liste plus restreinte de contacts

Critères de recherche:
  • Nom: On peut saisir un nom complet ou partiel pour cela terminer la chaîne par le caractère '*' ("B*" permet de chercher tous les contacts dont le nom commence par 'B')
  • Prénom: On peut saisir le prénom complet ou partiel pour cela terminer la chaîne par le caractère '*'
  • Pseudonyme: On peut saisir le pseudonyme complet ou partiel pour cela terminer la chaîne par le caractère '*'
  • Ville: On peut saisir la ville complete ou partielle pour cela terminer la chaîne par le caractère '*'
  • Nature: On peut choisir entre Homme, Femme, Société
  • Nature Contact: On peut choisir entre Famille, Amis, Connaissance, Professionnel
  • Remarque: Si aucun critère n'est modifié la recherche renvera la liste complète des contacts
On peut effectuer les actions suivantes:
  • Bouton Chercher: Lancer la recherche avec les critères saisis
  • Bouton Aide: Obtenir de l'aide sur la page
  • Bouton Retour: Retour à la liste des contacts
